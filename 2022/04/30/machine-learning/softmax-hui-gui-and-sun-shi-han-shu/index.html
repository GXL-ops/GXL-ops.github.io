2. 损失函数

不同的损失函数绘图

2.1 L2 Loss

蓝色是本函数，绿色是似然函数，橙色是损失函数的梯度

由上图可以看出损失函数的导数是如何更新我们的参数的。

因为Loss的导数决定我们如何更新我们的参数，当predictions<0时，导数为负数，所以更新参数靠右侧进行，反而反之，逐渐降低loss。

  1. 当真实值 $y$ 与预测值 $y’$ 距离较大时，可以知道Loss的单次更新是比较多的，梯度的绝对值会较大；
  2. 当真实值 $y$ 与预测值 $y’$ 逐渐靠近的时候，梯度的绝对值会变得越来越小，这就会使得网络层参数更新的幅度越来越小。

注意：有时真实值 $y$ 与预测值 $y’$ 距离较大时，我们并不想要大幅度更新参数，所以需要考虑L1 Loss

2.2 L1 Loss

当离原理距离比较远的时候，我们可能不需要特别大的梯度来更新参数，所以有 L1 Loss

导数是常数

真实值 $y$ 与预测值 $y’$ 距离不管多远，梯度值永远是常数，权重更新不会特别大，能够带来稳定性上的好处。

  • 稳定性好；
  • 零点处不可导；
  • 因为靠近零点的时候会出现震荡，所以优化到末期，不会很稳定。

2.3 Huber Loss

结合上面 L1 和 L2 的优势

真实值 $y$ 与预测值 $y’$ 距离的绝对值 <= 1的时候，梯度在变化的末期，能够逐渐平滑起来，梯度值逐渐降低，权重值更新变小。
